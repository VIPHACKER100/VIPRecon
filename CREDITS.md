## Technologies Used

VIPRecon is built using the following technologies:

### Core Stack
- **Python 3.8+** - Primary programming language
- **asyncio** - Asynchronous programming framework
- **aiohttp** - Async HTTP client/server library

### Key Dependencies
- **BeautifulSoup4** - HTML parsing and analysis
